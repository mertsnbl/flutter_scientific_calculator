## Setup and Run

```bash
flutter pub get
flutter run
```

Run on Chrome:

```bash
flutter run -d chrome
```

---

## Features

- Scientific calculator interface
- Basic arithmetic operations
- Trigonometric calculations
- Logarithm and square root operations
- Calculation history screen
